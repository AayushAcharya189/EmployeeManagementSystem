# Employee Management System

ASP.NET Core MVC application with role-based authentication, employee CRUD operations, soft delete, audit logs and admin dashboard.

## Features
- Identity Authentication & Roles (Admin, User)
- Role-based Authorization
- Soft Delete & Restore
- Search & Pagination
- Activity Logging
- Admin Dashboard
- Bootstrap Responsive UI

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Bootstrap 5
- Identity

## Setup
1. Clone repository
2. Update connection string in appsettings.json
3. Run migrations
4. Login as Admin
